Detailed Description

Create a set of agents that all have three places they know about, home, work and transport
.

First the home place is created, then agents, who all set this one place as home. Work places and transport vary by agent
Currently there are 3 agenst per home, 10 agents per work place, and 30 per tranport (a bus!)
If the travel schedule is set to stationary, however, only the home place will get used.
modelFactorySelector knows this as "simpleMobile".Use this class by creating a pointer to the sub-class:-

modelFactory* F=new simpleMobileFactory();

See modelFactorySelector
