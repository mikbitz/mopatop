Detailed Description

Simple static class to represent a very very simple disease.

Use of static class allows some of the details of the disease to be abstracted here, without needed a separate disease object in every agent
The disadvantage of this is that the parameters are fixed outside the class definition (a requirement of C++ for static (i.e. class-based rather than instance-based) variables)
but they can still be set in the constructor using a parameter file. Configure this once at the start of the code by creating an object and feeding it the parameterSettings object

parameterSettings parameters;
parameters.readParameters("../defaultParameterFile");
disease d(parameters);
