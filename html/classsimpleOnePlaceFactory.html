Detailed Description

Create a set of agents that all know only about one place, and remain there for all time, irespective of travel schedule
.

First the place is created, then agents, who all set this one place as home, work and transport. The latter two are set
in case at some point a different travel schedule is picked, in which case attemtps to move will all still just end up in the home location
modelFactorySelector knows this as "simpleOnePlace" . Use this class by creating a pointer to the sub-class:-

modelFactory* F=new simpleOnePlaceFactory();

See modelFactorySelector
