Detailed Description

Set up a wrapper class that will provide uniform pseudo-random numbers between 0 and 1
.

As usual with random number generators, the sequence is actually periodic with a very long period. By setting the seed one can generate the same sequence repeatedly by using the same seed. The C++ random number generators are rather complicated. This class allows selection of one of the available generators
without needing to look up the detail. It generates a random double between 0 and 1 using the mersenne twister generator.
The seed defaults to 0 but can be reset with setSeed - note this seems to need the twister to be completely re-created, hence the use of a pointer Example:-

randomizer r=randomizer::getInstance(0);
r.setSeed(12991);
double randomvalue=r.number();

Seems to be ok on multiple threads, but takes something of a hit from being only one RNG across them - would be faster if each
thread had its own RNG but this seems tricky to get right across multiple runs,
