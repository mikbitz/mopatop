Detailed Description

A class to pick one of a number of possible agent factories.

This is a static class used to define a pointer to a modelFactory
Each model factory can be selected using a name passed into the select method using
a string. This is used in the model class along with the parameterSettings to choose
what the model initialization of places and agents will look like.
