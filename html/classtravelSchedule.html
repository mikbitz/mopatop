Detailed Description

A simple fixed travel schedule that rotates cyclically between places.

The schedule is simply an ordered list of the types of place to be visited. The actual places are stored with the individual agents,
so that each agent can have a different location corresponding to a given type of place
Each agent sets up and stores its own copy of this schedule
Every time getNextLocation is called, the schedule advances one place forward. When the end is reached the schedule resets to the first entry.
