Detailed Description

originally a simple example demonstrating the non-interpolating fetch_points and fetch_values commands in MUI that are useful when just wanting to use MUI to pass data without using spatial interpolation.

Author
S. M. Longshaw, Mike Bithell
Date
25 September 2020,25/11/2021

MB 25/11/2021 updated the fetch for tranferring some agent data
Each data value transferred has an identifying point ID associated with it
Example run, with domains a and b (where a and b are examples of arbitrary user defined string - they must be different
but otherwise can take any value - for example instead of a nd b, we could use one and two, or 1 and 2.
mpiexec -f hosts -np 1 ./testagent parameterFilea a : -np 1 ./testagent parameterFileb b
Note that to get this to work on a local machine, a file called "hosts" may be needed -
this can just have the single value localhost in it. Without this MPI may get upset when trying to set the hostname
