Detailed Description

Places can have a list of occupants, and store disease contamination.

At the moment places are disjoint from each other - agents travel between them on a schedule, but don't interact with other not in the same place.
At present the interaction is indirect through contamination that agents in a given place can leave there, if they are carying a disease.
Places incliude mobile entities such as buses - so the schedule for movement between places needs to include explicit getting into vehicles, and a specifcation of time spent there,
as contamination level should depend on the duration of stay in a given place.

◆ update()

void place::update ( )
inline

The contamination in each place decays exponentially, or is reset to zero.

. This function should be called every (uniform) time step
This way places without any currently infected agents gradually lose their infectiveness, or else if
cleanEveryStep is set, the place has all contamination removed - useful if contamination shoudl only be present
as long as agents are present, and amount should be directly given by the number of agents. the decrement rate is assumed to be specified PER HOUR
