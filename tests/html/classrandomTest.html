◆ testSeed()

void randomTest::testSeed ( )
inline

test random seed behaviour

It is expected that the same random sequence is generated by the same seed. When the seed is reset
using the setSeed method, the random sequence should restart. Each call to the number() method should advance
the sequence by one step. The constructor should default to seed 0, but allow a seed to be set as an argument.
